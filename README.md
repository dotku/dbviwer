## GET START

1. modify the DB config (for MySQL only) in file /Common/Conf/config.php

eg.

>   return array(
>       'DB_TYPE'   => 'mysql', // 数据库类型
>       'DB_HOST'   => 'localhost', // 服务器地址
>       'DB_NAME'   => 'beecommute', // 数据库名
>       'DB_USER'   => 'root', // 用户名
>       'DB_PWD'    => 'admin123', // 密码
>       'PIN'   => '123456'
>   );

DONE <- that's all you need.
